# Study Notion - An EdTech Platform 📚🎓  

A full-featured educational platform built using the MERN stack, offering seamless learning experiences for students and instructors.  

## 🚀 Features  
- User authentication and Profile management (Students & Instructors)  
- Course creation and management  
- Video lectures and study materials  
- Payment integration via Razorpay  
- File uploads using Multer & Cloudinary for storage  
- Progress tracking for Students 
- Interactive Dashboard and data visualisation for Instructors
- Responsive UI  

## 🛠️ Tech Stack  
- **Frontend**: React, Tailwind CSS  
- **Backend**: Node.js, Express.js  
- **Database**: MongoDB  
- **Authentication**: JWT, bcrypt  
- **File Storage**: Multer, Cloudinary  
- **Payment Gateway**: Razorpay

## 📌 Installation & Setup  
1. Clone the repository:  
   ```sh  
   git clone https://github.com/tushar110302/Study-Notion-An-EdTech-Platform.git  
   cd Study-Notion-An-EdTech-Platform  
   ```
2. Install the dependencies:
    ```sh
    cd client && npm install  
    cd server && npm install  
    ```
3. Create `.env` file in client and server using the given `.env.sample`.

4. Start the development server both in client and server:
    ```sh 
    cd client && npm run dev
    cd client && npm run dev
    ```
